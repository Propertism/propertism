# Propertism Logo Redesign Guide

## Current Issues
- Outdated design with basic house icon
- Poor color contrast (gold/yellow on dark background)
- Not modern or premium-looking
- TM symbol placement is awkward
- Doesn't convey trust or professionalism

## Design Direction for New Logo

### Option 1: Modern Minimalist
**Concept**: Clean, geometric design with subtle real estate elements
- Simple "P" lettermark with architectural lines
- Monochrome or two-tone color scheme
- Sans-serif typography
- Negative space to create house/building shape

**Colors**:
- Primary: Deep Navy (#0A1628) or Charcoal (#2D2D2D)
- Accent: Gold (#C9A961) - used sparingly
- Alternative: White for dark backgrounds

### Option 2: Wordmark with Icon
**Concept**: Professional wordmark with minimal icon
- Clean, modern sans-serif font (like "Montserrat" or "Poppins")
- Small geometric icon (key, door, or abstract building)
- Horizontal layout for header
- Stacked version for mobile

**Typography**:
- Font Weight: 600-700 (Semi-bold to Bold)
- Letter spacing: Slightly increased for premium feel
- All caps or Title case

### Option 3: Badge/Emblem Style
**Concept**: Premium badge design (like luxury brands)
- Circular or shield shape
- "PROPERTISM" text around perimeter
- Central icon or monogram
- Vintage-modern hybrid style

## Recommended Specifications

### File Formats Needed
1. **SVG** - Scalable vector (primary)
2. **PNG** - Transparent background (1000x1000px minimum)
3. **PNG** - White version for dark backgrounds
4. **PNG** - Dark version for light backgrounds
5. **Favicon** - 32x32px, 64x64px

### Size Variations
- **Full Logo**: Horizontal layout with text
- **Icon Only**: Square version for social media
- **Compact**: For mobile header (smaller height)

### Color Variations
1. **Full Color**: Primary brand colors
2. **White**: For dark backgrounds (header)
3. **Black**: For light backgrounds
4. **Monochrome**: Single color versions

## Design Brief Template

When hiring a designer, provide this brief:

```
PROJECT: Propertism Logo Redesign

COMPANY: Propertism Realty Advisors
TAGLINE: Professional property management for NRIs

TARGET AUDIENCE:
- Non-Resident Indians (NRIs)
- Age: 30-55
- Income: Upper-middle to high
- Values: Trust, professionalism, modern service

BRAND PERSONALITY:
- Professional
- Trustworthy
- Modern
- Premium
- Approachable

DESIGN STYLE:
- Modern minimalist
- Clean and sophisticated
- Not corporate/stuffy
- Timeless (not trendy)

COLORS:
- Primary: Deep Navy (#0A1628)
- Accent: Gold (#C9A961)
- Alternative: White, Charcoal

MUST HAVES:
- Works on dark AND light backgrounds
- Scalable (looks good small and large)
- Professional and premium feel
- Easy to recognize

AVOID:
- Generic house icons
- Overly complex designs
- Dated/outdated styles
- Clipart-looking elements

COMPETITORS TO REFERENCE:
- Zillow (modern, clean)
- Compass Real Estate (sophisticated)
- Redfin (approachable, professional)

DELIVERABLES:
- Full color logo (horizontal)
- White version (for dark backgrounds)
- Black version (for light backgrounds)
- Icon only version
- All in SVG, PNG formats
- Favicon (32x32, 64x64)
```
